## 测试第一篇文章哦

使用Hexo部署一篇博客

```powershell
cnpm install -g hexo-cli
hexo init
hexo generate
hexo server
```

创建一篇博客的md文章

```powershell
hexo new "文章标题"
```

可以在md文章内指定日期，分类的信息哦~
